# Long 学生学习教育平台

> 面向中小学生的综合性学习教育 Web 平台

## 📋 项目简介

Long 是一个面向中小学生的综合性学习教育 Web 平台，提供学科知识学习、益智游戏、实用工具等功能。平台采用纯前端实现，无需后端服务器即可运行，适用于个人学习、课堂辅助、课外拓展等多种场景。

## ⚙️ 技术特点

- **纯前端实现**：无需后端服务器，打开浏览器即可使用
- **Tailwind CSS**：使用 Tailwind CSS 快速构建现代化界面
- **Font Awesome**：丰富的图标库，提升用户体验
- **响应式设计**：适配桌面、平板、手机等各种设备
- **丰富的 JavaScript 交互**：提供流畅的用户交互体验

## 🚀 使用说明

1. **打开主页**：双击 `index.html` 或在浏览器中打开该文件进入主页
2. **导航访问**：通过页面上的下拉菜单访问各类功能模块
3. **推荐入口**：使用 `MainFunction/功能导航.html` 查看整理后的所有功能（推荐）
4. **原始文件**：`Page/` 目录下为原始文件备份，可通过导航访问
